# 📊 Sales Scenario Planning Tool (Excel Model)

## 📌 Overview
This project is an Excel-based scenario planning tool used to analyze sales performance under different business conditions — Best Case, Realistic, and Worst Case.

It helps in understanding how changes in **sales volume, discount, and cost structure (COGS)** impact overall profitability and target achievement.

## ⚙️ Key Inputs
- Discount (%)
- Cost of Goods Sold (COGS %)
- Average Selling Price
- Sales Units
- Gross Margin Target

## 📈 Scenarios Covered
- Best Case (High Sales Volume)
- Realistic Case (Moderate Sales)
- Worst Case (Low Sales Volume)

## 📊 Metrics Calculated
- Net Invoice Sales
- Post Discount Revenue
- Net Sales
- COGS
- Gross Margin
- Variance vs Target

## 💡 Business Insights
- Helps identify risk in low-sales scenarios
- Shows impact of discounting on profitability
- Assists in decision-making for pricing and sales strategy

## 🛠 Tools Used
- Microsoft Excel
- Financial Modeling
- Scenario Analysis
